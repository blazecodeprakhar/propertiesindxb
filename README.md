# Dubai Luxury Villas Landing Page

A premium, state-of-the-art landing page for luxury villas and townhouses in Dubai. This highly optimized, mobile-responsive page is built strictly using modern HTML, CSS, and Vanilla Javascript. It is designed to capture high-quality leads and provide an exceptional user experience.

## ✨ Features
- **Modern Luxury Aesthetics:** Tailored color palettes, sleek dark overlays, and fluid micro-animations.
- **Dynamic CSS Carousel:** Showcases stunning high-definition renders of properties.
- **Dual-Form Lead Optimization:** Two highly intuitive lead generation forms with robust client-side validation.
- **Intelligent Phone Code Selector:** Automatically manages country dial codes and enforces digit-only input.
- **Premium Loading Transition:** A seamless, full-page transition screen with a pulsing gold spinner upon submission.
- **Relative Path Routing:** Completely compatible with subfolder deployments (e.g., GitHub Pages) and custom domains alike.
- **SEO & Conversion Optimized:** Structural semantic HTML, auto-fired Google Ads conversions, and Meta Pixel Lead event integration on the beautifully centered Thank You page.

## 🚀 Getting Started
1. Simply clone this repository:
   ```bash
   git clone https://github.com/blazecodeprakhar/propertiesindxb.git
   ```
2. Open `index.html` directly in any browser, or host it on your favorite web server.
